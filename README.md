## 编译

1. 安装 CMake 3.1 及以上版本
2. 进入`./bin/`文件夹
3. 执行`cmake ..` 生成对应平台的工程文件
4. 按照各个平台的编译方法编译

* Linux:执行`make`
* Windows:在VS控制台下执行`nmake`或者使用VS打开解决方案进行编译
* Mac OS X:使用Xcode打开工程文件编译
